## 大小端存储模式
### 大端：数据低位保存在内存的高地址中，
0x 12 34 56 78->0x 12 34 56 78
### 小端：数据低位保存在内存的低地址中
0x 12 34 56 78->`0x 78 56 34 12`

## char - 一个字节
| 补码 | 原码 | 十进制 |
|:-:|:-:|:-:|
| 0000 0000 |     ~     |   0  |
| 0000 0001 |     ~     |   1  |
| 0000 0010 |     ~     |   2  |
|    ...    |    ...    |  ... |
| 0111 1111 |     ~     |  127 |
| 1000 0000 |     \     | -128 |
| 1000 0000 | 1111 1111 | -127 |
| 1000 0001 | 1111 1110 | -126 |
|    ...    |    ...    |  ... |
| 1111 1110 | 1000 0010 |  -2  |
| 1111 1111 | 1000 0001 |  -1  |
